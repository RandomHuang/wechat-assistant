## 重要规则 - 操作确认

**用户要求：执行敏感操作前必须先确认**

### 需要确认的操作
1. Git 推送/提交（尤其是 push 到远程）
2. 删除文件/数据
3. 发送外部消息（微信、邮件等）
4. 修改系统配置（SSH、网络等）
5. 安装/卸载软件
6. 可能影响工作的命令

### 确认格式
```
⚠️  准备执行：[操作]
影响：[影响说明]
是否继续？(是/否)
```
